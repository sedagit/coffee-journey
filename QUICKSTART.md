# 快速開始指南 ⚡

## 5 分鐘上線你的咖啡網站!

### 📦 步驟 1: 解壓檔案
下載 `coffee-journey.tar.gz` 後解壓縮。

### 🔧 步驟 2: 上傳到 GitHub

**最簡單的方法(使用 GitHub 網頁界面):**

1. 登入 [GitHub](https://github.com)
2. 點擊右上角 "+" > "New repository"
3. Repository name: `coffee-journey`
4. 選擇 "Public"
5. 點擊 "Create repository"
6. 在新的 repo 頁面,點擊 "uploading an existing file"
7. 把解壓後的**所有檔案**拖拉進去(不要拖整個資料夾)
8. 在下方填寫 commit message: "Initial commit"
9. 點擊 "Commit changes"

### 🚀 步驟 3: 部署到 Netlify

1. 到 [Netlify](https://app.netlify.com) 註冊/登入
2. 點擊 "Add new site" > "Import an existing project"
3. 選擇 "GitHub"
4. 選擇你的 `coffee-journey` repository
5. 部署設定會自動載入,直接點 "Deploy site"
6. 等待 1-2 分鐘,完成!

### ✍️ 步驟 4: 設定 CMS 編輯器

1. 在 Netlify Dashboard,點擊你的網站
2. Site settings > Identity > Enable Identity
3. Registration 設為 "Invite only"
4. Services > Git Gateway > Enable Git Gateway
5. 回到 Identity,點擊 "Invite users"
6. 輸入你的 email
7. 檢查信箱,完成註冊

### 🎉 步驟 5: 開始使用!

訪問 `https://你的網站.netlify.app/admin`

登入後就可以開始新增咖啡店文章了!
